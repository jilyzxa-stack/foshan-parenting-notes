# 一、通用基础知识（佛山市级统一规则）

---

## 1.1 招生对象年龄要求

- **小学一年级：** 年满6周岁，即2020年8月31日（含当天）前出生，且未接受过义务教育的适龄儿童
- **初中一年级：** 小学应届毕业生（须在全国学籍系统已作毕业处理）
- **禁止重复入读：** 已入读小学或初中、已拥有相应学段学籍的学生不能重新报读。一经查实取消入学资格

## 1.2 招生生源分类及录取优先级

佛山市公办义务教育学校招生统一分为以下类型，按优先级从高到低录取：

|优先级|类型|定义|
|---|---|---|
|1|户籍生|具有该镇街户籍的适龄儿童少年|
|2|优教服务对象|军人/消防/烈士等特殊群体子女|
|3|政策生|符合佛府办〔2024〕6号规定15类条件之一的非户籍生|
|4|积分生|持有效居住证通过积分排名竞争学位的非户籍生|

**铁律：户籍生 > 政策生 > 积分生。所有区所有镇街无例外。**

## 1.3 政策生15类认定条件

依据：《佛山市人民政府办公室关于印发佛山市非户籍适龄儿童少年入读公办义务教育学校实施办法的通知》（佛府办〔2024〕6号）

**第1-14类（特殊群体，符合即认定）：**

- 第1类：现役军人、烈士、因公牺牲或病故军人
- 第2类：国家综合性消防救援队伍人员、政府专职消防员
- 第3类：公安、司法行政机关烈士/英模/因公伤残警察
- 第4-6类：进藏干部/殉职基层干部/见义勇为人士
- 第7-11类：领军人员/优粤卡A/B卡/优粤佛山卡A/B/C卡持卡人/博士后
- 第12类：华侨
- 第13类：台湾人士
- 第14类：本街道户籍人士（佛山市内跨区）

**第15类"双五"条件（与普通打工人最相关）：**

在所在区/镇街连续缴纳社保5年（60个月）+ 连续居住证明5年（60个月）+ 有合法稳定住址和就业。

## 1.4 "双五"条件详细规则

**"双五" = 社保连续60个月 + 居住证明连续60个月**

**社保要求：**

- 连续缴纳60个月（不是"累计"）
- 补缴不计入年限
- 断缴1个月即条件不成立

**居住证明（3种方式满足其一即可）：**

|方式|要求|特殊说明|
|---|---|---|
|居住证|连续有效签注60个月|断签即不连续|
|纳税记录|连续缴纳60个月|零税申报可计入；补缴不计入|
|营业执照|起算日前已登记+连续有效60个月|注销后重注册不计入此前年限|

**核心关键词：连续。断1个月即条件不成立。没有补救机会。**

## 1.5 积分入学通用规则

依据：《佛山市新市民积分制服务管理办法》（佛府办〔2023〕20号）

**计分体系：**

- 入学总积分 = 基础分 + 加分
- 减分指标和一票否决指标另计

**核心加分项：**

| 项目   | 标准       | 上限     | 注意                    |
| ---- | -------- | ------ | --------------------- |
| 房产   | 每平方米1分   | 最高144分 | 须提供不动产权证书；购房合同/发票不能积分 |
| 社保年限 | 按年限积分    | 参照镇街指引 | —                     |
| 居住年限 | 按居住证年限积分 | 参照镇街指引 | —                     |
| 学历   | 按等级加分    | —      | 国家承认学历                |
| 技能证书 | 按等级加分    | —      | 职业资格/技能等级证书           |

**同分排名规则：** 以首次在佛山市办理《广东省居住证》（含暂住证）的先后时间确定排名。办证越早排名越前。

**其他规则：**

- 夫妻只能一方申请，不可同时为同一孩子积分
- 材料须一次性提交，逾期不受理不补交
- 积分学位 = 总招生计划 - 户籍生 - 政策生的剩余（非预先划定）

## 1.6 "人户一致"通用定义

**情况一：** 新生与父（母）同一户籍 + 户主为父（母）+ 房屋全部产权属父或母（含父母共有）+ 房屋地址与户籍地址一致

**情况二：** 新生与父（母）、（外）祖父（母）同一户籍 + 户主为父（母）或祖父（母）+ 房屋产权属祖父（母）或与父母共有 + 地址一致

**不满足的情况（统筹安排）：** 集体户、空挂户、亲友投靠入户、迟入户、产权不属直系亲属等

## 1.7 购房与入学的关系（2020年后）

|事项|结论|
|---|---|
|购房能否直接当政策生|✗ 不能（2020年1月1日起取消）|
|购房能否积分加分|✓ 能（每㎡1分，最高144分）|
|购房合同能否积分|✗ 不能（必须不动产权证书）|
|购房是否保证入学|✗ 不保证|

**里水镇2026年方案原文："从2020年1月1日开始，购房不再作为政策生申请条件，但可作为积分入学的加分项目。"**

## 1.8 公民同招规则

- 公办学校和民办学校同步招生
- 学生可同时报公办和民办
- 最终只能选一所入读
- **在民办学校注册 = 自动放弃公办学位**
- 户籍生放弃后可统筹回公办（不能选校）
- 政策生/积分生放弃后不再安排

## 1.9 虚假材料处罚

|区域|处罚|
|---|---|
|通用|取消当年入学资格|
|禅城区（初中）|取消资格 + **三年内不受理入学或转学**|
|南海区|取消资格 + 积分清零 + **一年内不得申请**|
|顺德区|取消资格 + 涉嫌违法追究法律责任|

## 1.10 "长幼随学"通用规则

依据：佛山教基〔2023〕11号

- 多孩家庭弟妹可申请优先入读哥姐所在学校
- **是"优先"不是"保证"** ——学位不足时不一定满足
- 哥姐须在目标学校在读非毕业年级
- 摇号学校一般不接受长幼随学
- 需提供：两孩出生证 + 户口本

## 1.11 不服从统筹安排的后果

**所有区所有镇街统一规则：不服从统筹安排 = 视为放弃学位，不再另行安排。**

## 1.12 学区划分规则

- 学区划分仅限当年有效
- 每年可能因新建学校、招生容量、生源变化调整
- **请勿将学区作为购房或迁户的依据**（顺德区大良方案原文）
