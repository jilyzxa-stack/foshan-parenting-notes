### 版本A：口播讲解版（3-5分钟）

**【开头】**

各位佛山家长好，今天讲积分入学中居住证年限怎么算。

上一期讲了积分入学跟政策生的区别——最核心的就是"断了怎么处理"不同。今天把积分入学中居住证的计分规则讲透。

先给结论：积分入学中居住证断签了——断开的月份不算分，但前后有效时间可以连接累加。不像政策生那样"断了就全废"。

**【正文-计分规则】**

禅城积分制服务指南原文："居住时限每满1年积20分，最高可积240分。居住证（暂住证）未按要求续期的断开时间不计算，其余有效居住证（暂住证）时间可连续计算。"

翻译成大白话：

- 每满1年=20分
- 最高240分（12年封顶）
- 中间断了的月份不算——但前面和后面的有效月份加起来算

举例：你办了8年居住证，中间断了1年→实际有效时间=7年→积分=7×20=140分。不是断了就从头算，是扣掉断的月份其余累加。

**【正文-跟政策生的关键区别】**

| 规则   | 政策生条件         | 积分入学计分      |
| ---- | ------------- | ----------- |
| 要求   | 必须连续60个月      | 不要求连续       |
| 断签处理 | 断了=不满足条件（从头算） | 断开月份不计，前后累加 |
| 效果   | 全有或全无         | 有多少算多少      |

这是最核心的区别。政策生是"达标制"——差一个月就不是。积分入学是"累加制"——有多少算多少分。

所以如果你居住证中间断了——政策生不满足了，但积分入学还能算分。断了不代表全废。

**【正文-计算截止日】**

禅城原文："积分入学申请的居住年限计算时间截至申请年度的5月31日（含当日）。"

也就是说系统算你居住证年限时截止到5月31日。5月31日之前的有效月份都算分，之后的不算进当年。

如果你的居住证是5月到期——在5月31日之前续签就不影响当年计分。

**【正文-暂住证也算】**

原文特别写了"居住证（暂住证）"——如果你以前办的是暂住证后来转成居住证，暂住证的时间也算。

很多在佛山十几年的老居民早年办的是暂住证——这部分年限也计入积分。不要以为只有居住证才算。

**【正文-怎么查自己的居住证年限】**

方法一（线上）：微信小程序"粤居码"→居住证→居住证年限查询→选佛山→提交。

系统会显示你所有签注记录的起止日期——断了多少一目了然。

方法二（线下）：携带身份证和居住证到业务办理服务窗口查询。

建议：现在就去查一下。看有效月份合计多少→乘以20÷12≈你的居住证积分。

**【正文-怎么最大化居住证积分？】**

策略一：办得越早越好。每早办一年=多20分。同分时首次办证时间也是决胜因素。

策略二：不要断签。虽然断了不像政策生那样全废，但断签的月份不算分=白白损失积分。设年度续签提醒。

策略三：到期前30天续签。不要等到过期才去——过期后有一段"未签注"时间不算分。

策略四：如果已经断了——赶紧续签。断了的月份损失了但后面继续积累。不要因为"断了就全废"的误解而放弃续签。

**【结尾】**

总结：积分入学中居住证断签不像政策生那样全废——断开月份不算分但前后累加。每年20分最高240分。截止到5月31日。暂住证也算。

建议：第一现在就去粤居码查自己的有效年限。第二不要因为断了就放弃——断了前后还能累加。第三设年度续签提醒——每断一个月就少约1.67分。第四居住证办得越早越好——加分+同分决胜。

**一、计分规则（禅城积分制服务指南原文）**

要点拆解：

1. 每满1年=20分（即每满1个月≈1.67分）
2. 最高240分=12年封顶
3. 断签处理："断开时间不计算"=断的月份不加分
4. 累加规则："其余有效时间可连续计算"=断前+断后的有效月份相加
5. 截止日：积分入学=5月31日
6. 暂住证：原文括号注明"含暂住证"——以前的暂住证年限也算

|维度|政策生第15类|积分入学计分|
|---|---|---|
|计分/判断方式|达标制（连续60个月=通过）|累加制（有多少算多少分）|
|连续性要求|必须连续（中间不能断）|不要求连续|
|断签后处理|不满足条件=全废|断开不计但前后累加|
|补缴/重签后|重新开始计算连续|继续累加（新签注的月份照常加分）|
